#  "Разработка веб-приложения для удалённого мониторинга и управления парком 3D-принтеров"

Дипломная работа посвящена **"Разработка веб-приложения для удалённого мониторинга и управления парком 3D-принтеров"**. Сейчас во многих организациях процесс выглядит так: клиент присылает модель, сотрудник вручную рассчитывает стоимость, сам готовит файл для печати, запускает принтер по одному и параллельно ведёт учёт материалов в таблицах. Это долго, требует много ручной работы и часто приводит к ошибкам.
В проекте будет создана система, которая значительно упростит этот процесс.

## Как работает система:

1. **Загрузка модели.** Администратор загружает на сайт 3D-модель (файл 3MF).
2. **Автоматический расчёт.** Программа сама определяет, сколько материала понадобится, сколько времени займёт печать и рассчитывает стоимость.
3. **Проверка материалов.** Система проверяет, есть ли нужный материал в достаточном количестве. Если его не хватает — администратор получает уведомление.
4. **Подготовка к печати.** Если материала достаточно, система автоматически подготавливает управляющий файл для печати.
5. **Запуск печати.** Администратор может выбрать один или несколько принтеров, на которые отправить задание. Это позволяет:
   - Распределять большой заказ по парку принтеров;
   - Эффективно использовать оборудование и сокращать сроки изготовления.
6. **Удалённое управление.** Все 3D-принтеры подключены к серверу через специальное программное обеспечение-агент, установленное на Orange Pi. Агент принимает команды от веб-сервера (сайта), передаёт управляющий G-code на принтер и в реальном времени возвращает статусы печати (температуру, прогресс, ошибки). Благодаря этому администратор управляет парком оборудования через единый веб-интерфейс, не подключаясь напрямую к каждому устройству.
7. **Мониторинг.** Через сайт можно в реальном времени наблюдать за процессом: стадия печати, оставшееся время, а также видеопоток с камеры, если она подключена.
8. **Учёт материалов.** После печати система автоматически обновляет остатки материалов. Если запас падает ниже нормы, администратор получает уведомление о необходимости закупки.
9. **Статистика и аналитика печати:** Система собирает и визуализирует данные, которые помогут в управлении производственным процессом:
   - Расход материалов по периодам (день, неделя, месяц, квартал).
   - Затраты в разрезе типов пластика (PLA, ABS, PETG и т.д.).
   - Время работы каждого принтера (нагрузка, коэффициент использования оборудования).
   - Количество печати и их средняя стоимость.
   - Отказоустойчивость: процент неудачных печатей, причины ошибок (например, нехватка материала, сбой оборудования).

## Преимущества проекта:

- Экономия времени администратора и снижение количества ошибок.
- Централизованное управление всеми принтерами через сайт.
- Возможность одновременно печатать один и тот же файл на нескольких устройствах.
- Прозрачный учёт расходных материалов.
- Аналитика.
- Простое расширение: при подключении новых принтеров они автоматически становятся доступными в системе.
