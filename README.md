## Разработка веб-приложения для удалённого мониторинга и управления парком 3D-принтеров

Компания хочет создать программную систему для удалённого мониторинга и управления распределённым парком 3D-принтеров, используемых в производстве.

**Пользователи:** администратор, менеджер, технический специалист, потенциально сотни внешних заказчиков.

**Требования:**
1. Предоставить единый веб-интерфейс для управления всеми принтерами.
2. Автоматизировать коммерческие расчёты на основе анализа 3D-моделей, исключив человеческие ошибки.
3. Исключить простои из-за отсутствия материалов с помощью системы учёта и оповещений.
4. Дать возможность удалённо ставить задания, отслеживать прогресс и получать телеметрию (температура, видео) с любого принтера.
5. Собирать аналитику для обоснованных решений по оптимизации загрузки парка.
6. Обеспечить безопасный доступ и отказоустойчивость (сбой одного принтера не влияет на систему).

**Дополнительный контекст:**
1. Текущий процесс ручной, неэффективный и не масштабируется.
2. Создание моделей осуществляется самим пользователем, во внешней системе.
3. Система разворачивается на локальном сервере (Orange Pi) в сети предприятия, интеграция с внешними CRM/ERP не требуется на первом этапе.
